Replace backward slash '\' with forward slash '/' in vim

Ask Question
up vote 4 down vote favorite
1

I have some scripts(linux) which are generating the o/p with respect to Windows. I expect it to be /home/manty/... it generates \home\manty\... It is very annoying when I try to change it all the time. I am using gedit to find and replace. But in vim how can I do it? I tried the following but couldn't do it:

:%s/\\/\/

It changed only in the two places. I tried several other combinations as well nothing is working out.

Ah. I missed the g thing.

:%s/\\/\//g

This command does the trick.

I will wait for better answers from vim geeks. I am sure there must be a better way.
